Question ONE: Choose the correct answer:

1. The number of objects in a web page which consists of 4 JPEG images and HTML text is:
A. 1
B. 3
C. 5
D. 8
ANSWER: C
EXPLANATION: A web page object is a discrete file requested from the server. The base HTML text document counts as 1 object, and the 4 JPEG images count as 4 separate objects. Total: 1 + 4 = 5.

2. The values GET, POST, HEAD, etc., are specified in
A. Request & Response
B. Request
C. Response
D. None of The Above
ANSWER: B
EXPLANATION: GET, POST, and HEAD are HTTP Request Methods (or verbs). They are exclusively sent by the client within the initial Request line to tell the server what action to perform.

3. Find the oddly matched HTTP status codes:
A. 200 - OK
B. 400 - Bad Request
C. 301 - Moved Permanently
D. 304 - Not Found
ANSWER: D
EXPLANATION: HTTP 304 is Not Modified; Not Found is 404, so 304 - Not Found is the odd pair.

4. HTTP uses the services of ______ on well-known port 80:
A. TCP
B. UDP
C. SMTP
D. HTTPS
ANSWER: A
EXPLANATION: HTTP (Hypertext Transfer Protocol) relies on the Transmission Control Protocol (TCP) as its underlying transport layer to ensure reliable, ordered, and error-checked delivery of a stream of bytes.

5. In HTTP, the first line in a request message is called a ______ line; the first line in the response message is called the ______ line.
A. Request, Status
B. Status, Status
C. Response, Status
D. None of The Above
ANSWER: A
EXPLANATION: The initial line of an HTTP request is explicitly named the 'Request-Line' (containing the Method, URI, and HTTP Version). The initial line of an HTTP response is the 'Status-Line' (containing the HTTP Version, Status Code, and Reason Phrase).

6. An HTTP request message always contains
A. A Header and A Body
B. A Request Line and A Header
C. A Status Line, A Header and A Body
D. None of The Above
ANSWER: B
EXPLANATION: A valid HTTP request must have a Request Line. In modern HTTP (1.1+), at least one header (the `Host` header) is mandatory. The Body is optional (e.g., standard GET requests have no body). Therefore, 'Request Line and A Header' is the most accurate minimal requirement.

7. ______ is more robust and secure than ______
A. Get, Post
B. Post, Put
C. Post, Get
D. None of The Above
ANSWER: C
EXPLANATION: POST is considered more secure and robust than GET for transmitting sensitive or large amounts of data because POST places the data in the HTTP message body, whereas GET exposes data in the visible URL string, which is logged by servers and browser history.

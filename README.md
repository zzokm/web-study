# Web Study

Practice web app for **Web Technology** course finals (2021, 2024, 2025), with explanations and links to lecture slides.

**Live site:** [https://webtech.yehia.dev](https://webtech.yehia.dev)

## Features

- Practice by lecture or by exam year
- PDF viewers for frontend and backend lecture decks
- Exam analytics and repetitive-question highlights
- Saved questions (browser local storage)

## Quick start

### Local development

```bash
cd web
npm install
npm run dev
```

`predev` runs content sync automatically (`npm run sync` copies `data/` into `web/public/`).

Open [http://localhost:3000](http://localhost:3000).

### Docker

```bash
docker compose up --build
```

Serves the static export on port **7821** (mapped to container port 3000).

## Repository layout

| Path | Purpose |
|------|---------|
| [`data/`](data/) | Source JSON: exams, manifests, allocations, lecture PDFs, analysis |
| [`scripts/`](scripts/) | Python tooling for topic allocations on exam JSON |
| [`web/`](web/) | Next.js static app |

## Data workflow

1. Edit exam JSON or allocation maps under [`data/`](data/).
2. Apply lecture topic tags to exam questions (optional):

   ```bash
   python scripts/apply_topic_allocations.py
   ```

3. Sync into the web app:

   ```bash
   cd web && npm run sync
   ```

Do **not** edit `web/public/data/` directly — it is regenerated by sync.
